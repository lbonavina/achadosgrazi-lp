# AchadosGrazi

Economize em cada achado com a plataforma que reúne os melhores cupons e descontos do Brasil em um único lugar.

🔗 [Landing Preview](https://achadosgrazi.com.br/)

---

## 🎯 Visão Geral

AchadosGrazi é uma landing page para uma plataforma de cupons e descontos que reúne ofertas verificadas de grandes marcas nacionais. O objetivo é criar um canal único para consumidores se inscreverem e receberem avisos quando o site e os grupos exclusivos (WhatsApp e Telegram) estiverem no ar.

---

## ✨ Principais Funcionalidades

- Captura de e‑mails para cadastro de pré‑lançamento.  
- Mensagens claras sobre o propósito do site: cupons verificados, sem spam, apenas um aviso quando o site abrir.  
- Chamadas para ação simples e diretas para incentivar o usuário a deixar o e‑mail.  
- Previsão de grupos exclusivos no WhatsApp e no Telegram para acompanhar cupons em tempo real.  

---

## 📊 Destaques

- **Cupons verificados**: todos os códigos são testados antes de publicar.  
- **Sem spam**: o usuário recebe apenas um aviso quando o site estiver no ar.  
- **Centralização**: marcas do Brasil reunidas em uma única plataforma.  
- **Canais de comunicação**: grupos exclusivos no WhatsApp e no Telegram para acompanhar ofertas em tempo real.

---

## 🚀 Futuro do Projeto

Com o lançamento oficial, o AchadosGrazi deve:

- Disponibilizar uma área de navegação por categorias e marcas.  
- Implementar um painel administrativo para gerenciar cupons e ofertas.  
- Integrar cadastro e autenticação para usuários.  
- Oferecer opções de notificação por e‑mail, WhatsApp e Telegram segundo preferência do usuário.
